Movie API

Overview

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Getting Started

The current version of the API lives at https://github.com/CalebRD33/movie_api

Versions

Version 1: Initial Deployment 6/7/2024

Endpoints

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Navigate to homepage / GET None HTML page index.html
Navigate to documentation page /documentation.html use(express.static('public')) None HTML page documentation.html
Return a list of all movies to the user /movies GET None A JSON object holding data about all the movies
Return data about single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie
Return data about a genre description by genre name /movies/genre/[Genre Name] GET None A JSON object holding data about genre name and description
Return data about a particular director by name /movies/director/[Director Name] GET None A JSON object holding data about the chosen director
Allow new users to register /users POST A JSON object holding data about the new user to add, structured like:
{ "Username" : "Javid", "Password" : "password777", "Email" : "Javid@gmail.com", "Birthday" : "1989-02-19" } 		A JSON object holding data about the new user that was added, including an ID:
{ "Username": "Javid", "Password": "password777", "Email": "Javid@gmail.com", "Birthday": "1989-02-19T00:00:00.000Z", "FavoriteMovies": [], "_id": "6678724ba550664deda91945", "__v": 0 }
Allow users to update their user info /users/[Username] PUT A JSON object holding data about the info you would like to change:
{ "Password" : "password888" } 		A text message indicating the info has been updated
Allow users to add a movie to their list of favorites /users/[Username]/movies/[Movie ID] POST Place Username and Movie ID in the URL as follows: /users/Javid/movies/6671d2e911ecb9fd4890df00 A JSON object showing that the MovieID has been added to the FavoriteMovies array
Allow users to remove a movie to their list of favorites /users/[Username]/movies/[Movie ID] DELETE Place user ID and Title in the URL as follows: /users/Javid/movies/6671d2e911ecb9fd4890df00 A JSON object showing that the MovieID is no longer in the FavoriteMovies array
Allow existing users to deregister /users/[Username] DELETE Place username in the URL as follows: /users/Javid A text message that the user has been removed
